demo runner
=======================
A code runner for html/css/js;

Demo: http://demo.ruosen.io/demo-runner/

Press `cmd + s` or `ctrl + s` in editor to preview & save;


# Init

```
npm install
```
    
# Dev

start both webpack-dev-server and node server

```
npm run dev
```

# Build

```
npm run build
```

# Start Server

```
node server/app.js 
```

or using pm2
